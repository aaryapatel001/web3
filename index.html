Introduction to Variables

Python Variables & Types ⏱ 5 min read Basics

What are Variables?

A variable is a named container that stores a value in memory. In Python, you don't need to declare the type — Python figures it out automatically based on the value you assign. This is called dynamic typing.

Key Concepts

✦Dynamic typing — no type declarations needed in Python
✦Variables are references to objects stored in memory
✦Names are case-sensitive: myVar ≠ myvar
✦Use descriptive names: user_age is better than a
example.py
# Assigning different data types
name = "Alice" # str
age = 25 # int
height = 5.6 # float
is_student = True # bool
 
# f-string formatting
print(f"Name: {name}, Age: {age}")
# Output: Name: Alice, Age: 25

Multiple Assignment

Python allows you to assign multiple variables in a single line. You can also swap variable values without a temporary variable — something many other languages can't do as elegantly.

multiple_assign.py
# Multiple assignment
x, y, z = 1, 2, 3
 
# Swapping variables — Pythonic way
a, b = 10, 20
a, b = b, a
print(a, b) # 20 10
⚠ Watch out for these
✕ Starting variable names with numbers: 1name = "bad" — SyntaxError
✕ Using reserved keywords: if = 5 — will always cause SyntaxError
✕ Single underscore alone _ is valid but conventionally means "throwaway value"

Array Hash Table Python

Given an array of integers nums and an integer target, return the indices of the two numbers such that they add up to target. You may assume that each input would have exactly one solution, and you may not use the same element twice.

Examples
Example 1
Input: nums = [2,7,11,15], target = 9
Output: [0, 1]
Explanation: nums[0] + nums[1] == 9
Example 2
Input: nums = [3,2,4], target = 6
Output: [1, 2]
Constraints
· 2 ≤ nums.length ≤ 10⁴
· -10⁹ ≤ nums[i] ≤ 10⁹
· -10⁹ ≤ target ≤ 10⁹
· Only one valid answer exists.
🧠 Get AI Hint -50 pts
💡 Think about what information you need to store as you iterate through the array. For each number, what would you need to find its complement? A hash map can give you O(1) lookup time instead of O(n).

Valid Parentheses

Easy
Stack String Python

Given a string s containing just the characters '(', ')', '{', '}', '[' and ']', determine if the input string is valid. An input string is valid if open brackets are closed by the same type of brackets, and in the correct order.

Examples
Example 1
Input: s = "()"
Output: true
Example 2
Input: s = "()[]{}"
Output: true
Example 3
Input: s = "(]"
Output: false
🧠 Use AI Hint 1 remaining · -50 pts
💡 A stack is perfect here. Push opening brackets onto the stack. When you see a closing bracket, check if the top of the stack is its matching opening bracket. If the stack is empty at the end, the string is valid.
